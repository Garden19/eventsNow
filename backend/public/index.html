Welcome to the EventNowAPI

Please see the below points to guide you through the API.

What is this for?

The plan for this API is to be used alongside a frontend framework.

For event data to be gathered either by using external APIs or through scraping by using this API and then providing data to a frontend.

How can this be used?

Currently there is only the following endpoint:

/outdoor-cinemas
This endpoint will make a GET request which will then scrape outdoor cinema venues, showings and dates. After the data is scraped, it will then make a fetch request to the TMDB API to gather movie posters, images, titles, release dates and more. The data collected is excessive currently however I plan for it to be utilised later down the line as the development progresses.
